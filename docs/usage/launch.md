# Launch Hyprland

You can find the latest information how to launch Hyprland in the Hyprland Wiki.

https://wiki.hypr.land/Getting-Started/Master-Tutorial/#launching-hyprland

## From SDDM

The recommended way to launch the ML4W OS Hyprland is with the Display Manager SDDM.

## From TTY

You can also start Hyprland from the command line with

```sh
# Start Hyprland on 0.53.x
start-hyprland

# Start Hyprland on 0.52.x
Hyprland
```
